NOTA:

    Este projeto é um programa para determinar a combinação
    ideal de itens a serem colocados em um caminhão, 
    maximizando o valor total dos itens que se deve levar 
    para um evento beneficiente, sem exceder a capacidade do caminhão.


INSTRUÇÕES:

    1º  Certifique-se de ter instalado a linguagem de programação C e um compilador compatível em seu sistema;

    2º  Utilize o arquivo "entrada.txt" no mesmo diretório dos arquivos baixados. 
        Este arquivo contém os dados de entrada e pode ser modificado, 
        seguindo o seguinte formato:

            2.1  A primeira linha contém dois números inteiros separados por espaço: o número de itens e a capacidade máxima do caminhão;

            2.2  As linhas seguintes contêm informações sobre cada item. Cada linha contém dois números inteiros separados por espaço: o valor do item e o peso do item;

    4º  Execute o arquivo "evento_beneficiente.exe".

Decisão de Design:
	Foi optado uma abordagem recursiva do problema para encontrar a melhor solução;
	O projeto foi construído de maneira modular;
	O programa mede e exibe o tempo de execução para fornecer uma ideia do desempenho do algoritmo;
	Mede a quanntidade de chamadas recursivas necessárias para solucionar;
	O programa permite salvar o resultado final em um arquivo de texto.
